# Descrição

<p>
  Projeto desenvolvido para gerenciar o cardápio de uma confeitaria, 
  permitindo que seus clientes possam visualizar seus produtos, ou inseri-los
   em um carrinho para realizar futuras compra.
</p>

## Tecnologias Usadas

<ul>
 <li>NextJs</li>
 <li>SASS</li>
 <li>Apollo</li>
 <li>Vercel</li>
 <li>GraphCMS</li>
 <li>GraphQL</li>
</ul>
